Working directories

You can set the working directory of your document using the setwd() function. You can check the current working directory using getwd(). Alternatively, you can set the working directory manually in RStudio (Session > Set Working Directory).

Install packages

Packages can be installed from RStudio using the ‘Packages’ tab (normally in the bottom right window). Search for the package you want and hit enter. You can manually install packages from the console using the install.packages() function. Ex. install.packages('devtools').

Loading packages

To load a package after you have installed it, use the library() function. Try loading the tidyverse package.

Comments

In R, anything that comes after # is considered a comment. Comments are important because they help you to remember what you did, why you did it, etc. Add 2 + 2 and create a relevant comment:

#############################
# Always comment your code! #
#############################

# This is a comment
2 + 2

R as a calculator

This isn’t incredibly useful, but it gives you an idea of how R works. Try some simple math:

#####################
# R as a calculator #
#####################

2 + 2

4^2

(12 * 15) / 2

Assigning variables

R uses objects. You must assign them using <- in order to store them in memory. You can them call them by typing the variable or calling print(). Store the value ‘2’ to the object ‘x’

x <- 2
print(x)

# Perform an operation on 'x'
x + 4
